技术选型

前端框架React 18 + TypeScript + Vite
路由React Router v6
状态管理Zustand（轻量、无模板代码）
样式方案Tailwind CSS v4（暗色主题 + 氛围感优先）
推理画布React Flow（节点连线拖拽）
动画Framer Motion
后端Express + TypeScript + tsx
AI SDK@anthropic-ai/sdk
数据存储文件系统 JSON（服务端 data/ 目录）

数据流

1️⃣ 前端选择参数 → POST /api/generate
↓
2️⃣ 后端 Prompt 编排流水线（骨架→人物→对话→校验）
↓
3️⃣ 输出结构化 CaseData JSON → 存入 server/data/ → 返回前端
↓
4️⃣ 前端 Zustand Store 加载案件数据，驱动各个页面渲染
↓
5️⃣ 玩家操作 → Store 更新（收集线索、对话进度、推理状态）
↓
6️⃣ 提交推理 → POST /api/verify → 比对真相 → 返回评分

核心类型定义预览

interface CaseData {
  meta: { title, style, difficulty, type, summary }
  victim: { name, age, occupation, causeOfDeath, timeOfDeath, location }
  characters: Character[]
  scenes: Scene[]
  clues: Clue[]
  truth: { culprit, motive, method, timeline, keyEvidence, twist }
}

interface Character {
  id, name, age, occupation
  relationToVictim, surfaceMotive, hiddenSecret
  alibi, telltaleLie
  dialogues: Dialogue[]
}

interface Dialogue {
  text, speaker, phase // phase: "initial" | "deep" | "confrontation"
  options?: { text, leadsTo, unlocksClue? }[]
}

interface Clue {
  id, name, description
  type: "physical" | "testimony" | "contradiction"
  foundInScene, relatedCharacter
}
